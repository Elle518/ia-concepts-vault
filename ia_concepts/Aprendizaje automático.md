## Definición

El **aprendizaje automático** o *machine learning* (ML) es una subárea de la [[Inteligencia artificial|inteligencia artificial]] en la que un sistema aprende patrones a partir de datos en lugar de ser programado explícitamente con todas las reglas.

El objetivo es que el modelo generalice a ejemplos nuevos: por ejemplo, clasificar una imagen no vista, predecir ventas futuras o agrupar clientes por comportamiento.

## ML vs programación tradicional

En programación tradicional escribimos reglas: "si ocurre X, haz Y".

En ML damos ejemplos y dejamos que el algoritmo ajuste un modelo. En vez de codificar manualmente la regla para distinguir perros y gatos, entrenamos con muchas imágenes y etiquetas para que el modelo aprenda regularidades útiles.

## Tipos de ML

Un sistema de ML aprende una función aproximada a partir de datos. Según el tipo de señal disponible, se suele dividir en:

- [[Aprendizaje supervisado]]: los datos incluyen etiquetas o respuestas esperadas.
- [[Aprendizaje no supervisado]]: los datos no incluyen etiquetas y el objetivo es descubrir su estructura o patrones ocultos en los mismos.
- [[Aprendizaje autosupervisado]]: las etiquetas se generan a partir de los propios datos.
